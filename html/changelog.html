14 February 2023

RKz, lizardqueenlexi updated:

  • Alternative steps will now show on Operating Computers! We technically have had this feature for the last 3 years, it just wasn't displaying correctly.

SinguloMirrorbot updated:

  • Motorized wheelchairs now drain power cells as intended instead of dozens of times per second when a directional key is held down.
  • Motorized wheelchairs now play a soft beep and alert when the installed power cell reaches 25% charge
  • deadchat will be sent to discord as well as long as a round ended.
  • New config line - discord_ooc_channel. used to determine which discord channel OOC messages will be sent to.
  • Sending OOC message to discord now includes dead chats (after round finished only). OOC Discord message will describe whether the message is OOC or deadchat, and which server it was sent from.
  • fixed quirk resetting issue in a certain situation
  • Ability icons no longer go blank after cooldown completion
  • Fixed multiz movement exploit.
  • Added logging to purchases and uses of Malf AI modules.
  • Adds package sorting disposals pipes to RPDs and disposals dispensers. These sort wrapped and unwrapped objects.
  • fixed some code that didn't use random pill shape correctly
  • weirdly implemented pill image folder
  • medicine_containers image asset that is used by chemmaster tgui. old pill asset list is removed.
  • chem master will now use more wider pill & patch options
  • removed unused pill image (pill0) which is actually a duplication of a pill image(pill18). also arranged pill image order.
  • angry pill, sad pill, and question pill (random pick one)... new pill colours - pink, indigo, red, orange, etc... and pill example dummy image.
  • blank patch sprites
  • Adds current map to metric tracking
  • fixed a bug where broodmothers did not kill certain mobs when feeding on them
  • fixed an exploit that allowed broodmothers to feed on the same target multiple times
  • Broodmothers now drain all blood from their victims when feeding.
  • fixed some RPD code
  • added more debug items
  • made debugging items better
  • made the debug outfit better
  • changed some item paths from 'obj/.../admin' to 'obj/.../debug'
  • admin RCD is now ranged
  • New sprites for the delimber and hallucination anomalies

09 February 2023

Pigeonverde322, Imaginos16 updated:

  • new icons for materials and ores.
  • refactored how the material code is sorted out.
  • removed some unused material and ore icons that they where just bad.

SinguloMirrorbot updated:

  • Fix monkey clothing not generating on strip menu interactions
  • Added TGUI inputs and alerts internally, however they are not enabled.
  • Improved the response of most alert menus when you X them out rather than hitting No/Cancel.
  • Increased the contrast of selected/hovered TGUI buttons to 30% (from 15%)
  • Converted the message monitor to TGUI
  • The Message Monitor can now send admin messages with emojis and images.
  • Changed the flavor text for the monitor decryption key paper.
  • Emagging the message monitor no longer immediately prints the key, but instead prints the key when the message monitor is usable again (after 10 * keylength seconds). The key couldn't be used prior to this previously, so it just makes it more consistent.
  • Fixed PDA messages with images not always properly line breaking inside "View Messages" on the Messenger app.
  • Fixed a possible hard-del when the message monitor's linked server was deleted.
  • Added logging and admin messages for deletion of PDA/request logs.
  • Improved the Message Monitor's server linking behavior to support multiple servers of the same network and name.
  • Fixed an exploit where you could unlink the active message monitor server, link to another with a different decryption key, and still be authenticated.
  • Added an admin verb, "Send PDA Message"
  • quirk saving is hotfixed
  • Added cloth recipes craft time
  • Added the mail bag and bandana to cloth recipes
  • fixed the double bedsheet crafting output and empty sandbag icon
  • fixed durathread recipes using cloth recipes, somehow
  • Added name and job searching to the PDA Messenger contacts panel.
  • PDA message inputs are now TGUI-based.
  • When inputting a new PDA ringtone, the old one will be in the input text box already.
  • Improved some PDA messenger code.
  • TGUI text inputs now properly autofocus text.
  • Adds a desk bell, so you can let the service industry know how much you truly care about their hearing, alongside a new distinct sound for it!.
  • fixed techfloor_grey icon name

08 February 2023

Floyd,ActionNinja,BetaCentari updated:

  • added a TON of new sounds under sound/surgery and sound/items/handling
  • Changed some inventory handling to configure picking up and putting down and dropping items

Kit_Katz, RKz updated:

  • Added new customization options for apids. Color is now selectable from a pallette, and three customization options have been added for stripes, antenna, and bodymarkings
  • entirely new APID sprites for sprite_accessories, including stripes, antenna, and bodymarkings

SinguloMirrorbot updated:

  • Fix flammable trait not activating
  • Fix haunted activation.
  • Fix blacklisting on explosive malfunction
  • Add limit to clone & bear malfunction.
  • IPCs who become changelings now properly have their medical records updated and no longer spawn bald
  • The MK-II Ripley isn't completely inferior compared to the Firefighter - there's a reason to make it now! You should see more MK-II's after if this change goes through which is (probably?) good.
  • doubled the extrapolation speed of the Virus Extrapolator
  • Adjusted spider stats with a focus on reducing their ability to snowball through sheer numbers and increasing their reliance on venom instead of brute force.
  • Spiders are now immune to the vacuum and cold of space.
  • Broodmothers can no longer heal, but have more venom, damage and health
  • Nurses can no longer reproduce, but have slightly more health
  • Guards are now the slow tanky choice instead of tarantulas.
  • fixed an exploit with spider egg clusters and enriched spawns
  • Spiders are no longer available for science department traitors.
  • Farting now has a cooldown.
  • Added a role preference for Pirate Crew polls (it previously used the Traitor preference). Be sure to enable it!
  • Added Swarmers to Dynamic's midround pool.
  • Added Morph to Dynamic's midround pool.
  • Dynamic Nightmare will now properly use its new role preference.
  • Sentience potions, morph, nightmare, and swarmers random events will now properly poll using their preference rather than xenomorphs.
  • Fixed Space Dragon not being included in the default antagonist ban role list.
  • Fixed Morphs always receiving the "non antag" notification despite being antag.
  • Bans for swarmers will now prevent taking empty swarmer shells.
  • Bans for Space Pirates will now prevent taking pirate sleepers.
  • Cleaned up a lot of unnecessary code in Dynamic.
  • Adds a function to curtains - blocking light.
  • New monkey sprites
  • Tweaks monkey inventory update - procs interface with ``monkey_icon``
  • Add ``i_clothing`` slot to monkey equipment slots
  • Allow monkeys to equip ``i_clothing``
  • Add Experiment gimmick job
  • Add Experiment job-spawn to MetaStation
  • Space Dragons are no longer forced to open rifts within a timer.
  • Space Dragons can continue creating rifts, and no longer receives a slowdown if one is destroyed while charging.
  • If the Space Dragon is qdeleted without dying, it will now register as a loss for the rifts.
  • Fixed the Space Dragon roundend report showing a double "Space Dragon was assisted by"
  • Added a new malnourished bear type which is overall weaker than the normal one
  • Tweaked the bear malfunction of artifacts to spawn malnourished bears
  • Station traits are now chosen with more equalised randomness.
  • Added research for a Suit Storage Unit circuit board
  • Added the ability to build and deconstruct Suit Storage Units
  • Added machine parts to the Suit Storage Unit
  • Added the ability to crowbar open and pry the items out of a depowered Suit Storage Unit on harm intent
  • Suit storage unit's decontamination damage is lower by default when unhacked
  • Add dedicated DMI for monkey icons
  • Fixes missing icons for facehuggers - for monkeys
  • Tweaks monkey sprite to look less dumpy
  • Added the "Look Up" and "Look Down" Multi-Z verbs, with keybinds L (up) and ; (down).
  • Added zmove keybinds, F for up, C for down.
  • Added the Combustion Jetpack crate, an engineering crate that includes the combustion jetpack, which can move up and down in gravity as well as function in space like a normal jetpack. It burns a plasma-oxygen mix and is refillable (ideally at a ~50% ratio).
  • Added z-level changing buttons to the advanced camera console.
  • Zfalls now do more damage per level fallen, and felinids falling more than one level will still take damage, but reduced. Damage is also mostly applied to legs and split between the rest of the body in the case of missing or broken legs.
  • Refactored multi-z sound code slightly.
  • Using an RCD on open space (above another z-level) will now place floors.
  • Placing lattices on open space no longer requires pixelhunting.
  • Tweaked failure messages for Move Upwards/Move Down verbs.
  • Fixed zfalls stating that you fell one more z-level than you actually did.
  • Fixed sound breaking when changing z-levels.
  • Fixed various z-level changes not registering.
  • You can no longer move up and down through catwalks, and you also cannot "float" up while in gravity.
  • Fixed the move up/down animations being reversed.
  • Removed a broken old deconstruction hint for catwalks.
  • Fixed a runtime caused by zfalls.
  • Fixed zfalls stopping midair after one level.
  • Fixed AI camera static not updating when you move up and down.
  • Fixed the removal of a lattice or catwalk under someone not triggering a zfall.
  • Added jetpack and wing multiz flight. Regular jetpacks are not capable of enough thrust to resist gravity.
  • Moving up and down in space only takes 1 second, while in-gravity flight, etc take 3 seconds.
  • Slightly optimized gravity code for ztrait-style gravity.
  • Added the "rocket jetpack", a jetpack designed for in-gravity flight, that burns plasma and oxygen.
  • Fixed nightmare multiz jaunting
  • Falling is no longer instant, there is a period of 0.2s between zlevels (you can even move onto a solid tile to "break" your fall)
  • You can now move during all types of zmove except for climbing (which is disabled currently).
