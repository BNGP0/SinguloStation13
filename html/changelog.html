14 February 2023

RKz, lizardqueenlexi updated:

  • Alternative steps will now show on Operating Computers! We technically have had this feature for the last 3 years, it just wasn't displaying correctly.

SinguloMirrorbot updated:

  • Motorized wheelchairs now drain power cells as intended instead of dozens of times per second when a directional key is held down.
  • Motorized wheelchairs now play a soft beep and alert when the installed power cell reaches 25% charge
  • deadchat will be sent to discord as well as long as a round ended.
  • New config line - discord_ooc_channel. used to determine which discord channel OOC messages will be sent to.
  • Sending OOC message to discord now includes dead chats (after round finished only). OOC Discord message will describe whether the message is OOC or deadchat, and which server it was sent from.
  • fixed quirk resetting issue in a certain situation
  • Ability icons no longer go blank after cooldown completion
  • Fixed multiz movement exploit.
  • Added logging to purchases and uses of Malf AI modules.
  • Adds package sorting disposals pipes to RPDs and disposals dispensers. These sort wrapped and unwrapped objects.
  • fixed some code that didn't use random pill shape correctly
  • weirdly implemented pill image folder
  • medicine_containers image asset that is used by chemmaster tgui. old pill asset list is removed.
  • chem master will now use more wider pill & patch options
  • removed unused pill image (pill0) which is actually a duplication of a pill image(pill18). also arranged pill image order.
  • angry pill, sad pill, and question pill (random pick one)... new pill colours - pink, indigo, red, orange, etc... and pill example dummy image.
  • blank patch sprites
  • Adds current map to metric tracking
  • fixed a bug where broodmothers did not kill certain mobs when feeding on them
  • fixed an exploit that allowed broodmothers to feed on the same target multiple times
  • Broodmothers now drain all blood from their victims when feeding.
  • fixed some RPD code
  • added more debug items
  • made debugging items better
  • made the debug outfit better
  • changed some item paths from 'obj/.../admin' to 'obj/.../debug'
  • admin RCD is now ranged
  • New sprites for the delimber and hallucination anomalies
