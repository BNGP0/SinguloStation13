22 December 2022

MaltVinegar updated:

  • map-fix nuclear option

SinguloMirrorbot updated:

  • Several new admin smites
  • Newscasters now use TGUI and have new features like channel editing and descriptions.
  • Security Newscasters have been removed in favor of ID scanning on all newscasters.
  • Added Bounty Board to all newscasters, where you can post bounties for the crew to fulfill.
  • Reworked admin newscaster panel and topics to use TGUI.
  • Fixed some issues with IE8 compat classes on TGUI Flex components.
  • Fixed TGUI Buttons with right-positioned icons lacking a margin from the content text.
  • Added a Newscaster & Bounty Board ModPC app and added it to the Curator, HoP, and Captain's PDA roundstart, also downloadable.
  • Removed the Corginator from Xenoarch because the corgium trait was removed
  • Airlock, tile, and decal painter inhand sprites work correctly when facing sideways.
  • Removed a duplicate paper icon.
  • Added a variety of new greps to the Github and corrected map issues that failed the greps
  • Corgium will no longer last forever and will no longer affect people with damaged livers forever.
  • Autolathes no longer give the item they were processing if the required materials are removed during processing.
  • added the botany chem dispenser to research node and it can now be printed if researched
  • Refactored and abstracted the admin ticketing system for use by mentors
  • Added a mentor ticketing panel
  • Mentorhelps can now be escalated to adminhelps
  • Adminhelp de-escalation to mhelp button is now automatic and will create a corresponding ticket
  • Admins can now see mentorhelp tickets in the admin ticket manager
  • Removed unused mentor code
  • Admin Ticket Manager now has collapsible Closed and Resolved sections
  • All close actions on tickets now have a confirmation prompt
  • Fix a runtime caused by admin-following from the ticket panel for a player that has disconnected.
  • Overriding a claim on a ticket no longer shows the admin's IC name in the admin log, but their CKEY instead.
  • The ticket menu no longer shows a Re-open button if the ticket is open.
  • Improved the UI of the ticket messaging box and added auto-scroll when a new message appears.
  • Fixed issues with claiming and ticket resolve actions - it will now silently claim the ticket before resolving (and no longer messages the ticket subject about the claiming).
  • All ticket close actions except Resolve now have confirmation states in the ticket manager.
  • Adminhelp/mentorhelp resolve messages will now go under the "Admin PM"/"Mentor PM" chat filters.
  • Fixed admin PM replies sometimes creating HTML entities (like `'`)
  • Corrects the savefile importer so it properly loads equipped gear into the database

san7890 updated:

  • For developers, made the Merge conflict marker more explicit than before.

  • Added Radproof shutters next to the SM on all the maps.

  • fixed incongruence in price between constructing floor on lattice and constructing on the space below the lattice
  • Lighteaters will no longer erase hardsuits with headlamps from existence, instead, permanently disabling their headlamps.
