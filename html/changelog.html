  • Fixes polls not properly displaying
  • Cosmic space bedsheet is now a movable light
  • Revenants can no longer rest because it is pointless to do so as a revenant. Coincidentally revenants will no longer be able to accidentally turn off their ability to travel between Z-levels on glowstation because this is how they were doing that.
  • Re-added the Clown's ability to upload a honk virus to airlocks with their PDA.
  • Tower caps - [50% carbon]
  • Steel caps - [20% iron]
  • Rice - [10% Silicon]
  • Combustible Lemon - [10% sulfur, 10% charcoal 10% Saltpeter]
  • Redbeet - [2% Sodium]
  • Tea aspera - [2% Fluorine]
  • Tea astra - [2% Fluorine]
  • Poppy - [15% Morphine]
  • Moonflower - [8% Acetone]
  • New Watermelon mutation > Ballolon - [20% Oxygen, 20% Hydrogen]
  • New Grass mutation > Shamrocks - [10% nitrogen]
  • New 0.001% for a four leaf clover from a Shamrocks harvest
  • Sprites for Shamrock, clover and Ballolon (growing.dmi, growing_fruits.dmi, harvest.dmi, seeds.dmi, items_and_weapons.dmi)
  • The boop emote utilized by IPCs/silicons now has a noise
  • Fixes a runtime with security records consoles
  • Added a new subtype of labcoat used by doctors, and shifted the CMO, paramedic and brig physician typepaths into this subtype
  • All doctor labcoats can now store medkits in their suit storage slot
  • Medkits are now bulky items that cannot be stored in bags, with exception to the syndicate combat medkit.
  • Paramedics can now add the EMT Labcoat from the beecoin shop to their loadout. Geneticists and Chemists have lost access to this loadout option.
  • Heartbreaker is no longer always murderbone, instead being non-lethal date ruining 85% of the time.
  • Valentine's messages are now big and bold for clarity.
  • Fixed valentine's protect objectives always failing.
  • Fixed valentine's not setting players with the "in love" status effect.
  • Fixed valentine's not properly getting the gender of the described player in greentexts.
  • You can no longer select yourself as a valentine.
  • Non-crew can no longer be valentines.
  • Fixed valentine's not properly assigning silicon laws.
  • Clarified the valentine's announcement, saying that you can choose a valentine with the card.
  • Valentine's cards no longer require you to use a pen, instead just using it in hand to select your valentine.
  • You can now cancel your selection of a valentine.
  • Removed particularly sexual or LRP valentine's messages.
  • Added an antag HUD icon for the valentine's antagonist.
  • Added an antag HUD icon for the heartbreaker antagonist.
  • Lizard tails no longer ignore dna when regenerating organs.
  • Added the Durathread Jumpskirt, which shares the armor values with the Durathread Jumpsuit but doesn't cover up the legs
  • Added the crafting recipes for jumpskirts out of Cloth and Durathread
  • Capitalized the crafting recipe for the hoodie
  • Checks for role bans, ghost role toggles and ghost role cooldowns have been moved from spawners to client code for easier use in determining ghost role eligibility
  • Spider eggs, carp portals and other playable mobs now respect role bans, ghost role toggles and ghost role cooldowns.
  • fixed a variable typo in viper spiders - they can no longer lay webs instantly.
  • Removed one way to crash the game by admins using Fat sucker
  • BYOND member variant-ghosts can now display their accessories just like normal ghosts. No Corgi's though, sorry Bacon.
  • added a missing comment in bank account variable
  • GLOB.exploration_points is removed, and replaced by account system
  • ID card no longer holds mining points.
  • Bank account now holds mining points, and other exceptional currency like exploration points.
  • exploration points are now saved into sci budget account.
  • Shared Mining Account budget for golems.
  • Lavaland golems will get a mining card when they spawn. All golem cards will have "Shared Mining Account" instead of having their personal account for each golem, and mining points will be shared as well.
  • new vv to ID card - add mining points.
  • new vv to exploration vendor - add exploration points.
  • re-path'ed some unclear paths (mining id card is now pathed as /golem)
  • Added the sensible suit to the Curadrobe
  • wild magic apprentice will not learn soul tap/binding soul spells
  • wild magic apprentice will not learn the certain spells again
